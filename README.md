# MyD3DProject

## Run

```bash
MyD3DProject.exe {param}
```

### Params

| Params   | Description                |
| -------- | -------------------------- |
| -ins     | choose one instance to run |
| -geoType | choose a shape to render   |



#### -ins

* 0: Run InitializeApp
* 1: Run BoxApp

#### -geoType

this param for BoxApp only

* 0: BoxShape
* 1:pyramidShape
